# source

## Назначение

Входные документы-источники IOC. Дезорганизованные, часто экспортированные из
офисных форматов (Word `.htm`/`.docx`), с defanged-индикаторами (`hxxp[:]//`,
`[.]`, `[:]`).

**Правило слоя:** только входные данные. Чтение — через порт `SourceReader`
(адаптер на Apache Tika), формат определяется автоматически.

## Заметки

Tika определяет формат и кодировку; путь к источнику задаётся флагом
`--source`. Поддержка новых форматов — без изменения кода (Tika) или новым
`SourceReader`-адаптером.
