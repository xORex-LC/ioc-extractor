# com.iocextractor.domain.refang

## Назначение

Деобфускация (refang): возврат defanged-индикаторов (`hxxp`, `[.]`, `[:]`) к
разбираемому виду.

**Правило слоя:** чистая функция над текстом, без I/O. Порт `Refanger`
реализуется здесь же дефолтной реализацией; правила приходят из конфигурации.

## Структура

| Файл | Назначение |
|---|---|
| `Refanger.java` | Порт: `String refang(String)` |
| `ReplacementRefanger.java` | Упорядоченные литеральные замены (порядок важен) |
| `RefangRule.java` | Правило замены `from → to` |

## Заметки

Замены литеральные (не regex). Порядок значим: `hxxps→https` раньше `hxxp→http`.
