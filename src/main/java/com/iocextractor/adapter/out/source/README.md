# com.iocextractor.adapter.out.source

## Назначение

Выходной адаптер порта `SourceReader`: извлечение текста из документа.

**Правило слоя:** реализует порт; инкапсулирует Apache Tika. Сбои чтения/парсинга
не утекают наружу — оборачиваются в `IocExtractorException`.

## Структура

| Файл | Назначение |
|---|---|
| `TikaSourceReader.java` | Tika: автоопределение формата (.docx/.htm/.pdf/…) |

## Заметки

Лимит контента снят (`-1`), иначе Tika усекает текст на 100k символов. Новый
формат — обычно «бесплатно» через Tika, иначе новым адаптером порта.
