# com.iocextractor.common

## Назначение

Минимальные общие типы, разделяемые слоями (на текущем этапе — корневое
исключение).

**Правило слоя:** только нейтральные, бесфреймворковые типы без бизнес-логики.
Кандидат на вынесение в `platform/*`-модуль при переходе к многомодульности.

## Структура

| Файл | Назначение |
|---|---|
| `IocExtractorException.java` | Корневое unchecked-исключение для несбоев I/O/обработки |

## Заметки

Полноценная модель ошибок и трансляция на границах — отдельная сквозная
подсистема, см. `docs/cross-cutting.md`.
